# Azure Naming Generator

## Purpose

Azure Naming Generator is a web application that helps Azure administrators, cloud engineers, and DevOps engineers generate consistent and standardized Azure resource names based on organizational naming conventions.

The tool reduces manual effort, improves governance, and helps ensure resources follow naming standards across environments.

## Live Demo

https://mendore85.github.io/azure-naming-generator/

## Features

* Generate Azure-compliant resource names
* Support for multiple Azure resource types

  * Storage Accounts
  * Key Vaults
  * Resource Groups
  * Virtual Machines
  * App Services
* Copy generated names to clipboard
* History tracking
* Persistent history using Local Storage
* Clear history functionality
* Dark/Light mode
* CSV export of generated names
* Azure naming validation rules
